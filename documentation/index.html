Mr.Art — Mosaic Artist Website

Welcome to the official documentation for the **Freelance Mosaic Artist Portfolio**. This premium, single-page interactive website is designed for artists who want to showcase their craftsmanship with an elegant, mosaic-inspired aesthetic. Built with performance and responsiveness in mind, this project is ready for immediate deployment.

Key Features

  • Adaptive Mosaic Background with Floating Tile Animations.
  • Fully Responsive Dark/Light UI Support.
  • Interactive Service Cards with 3D Tilt Effects.
  • Filtered Portfolio Gallery with Image Lightbox.
  • Client Success & Process Walkthrough Sections.

1. Installation Guide

Setting up your new portfolio is straightforward. Follow these steps to get your site live on your local machine or hosting server.

Download and Extract

Extract the provided .zip file into a directory of your choice. Ensure you have all the core files: index.html, login.html, and register.html.

/freelance_mosaic_artist
├── index.html        (Main Website)
├── login.html        (Client Portal Login)
├── register.html     (New Client Signup)
└── assets/          (Images and Icons)

Open in Code Editor

We recommend using Visual Studio Code (VS Code). Open the project folder in VS Code to see the organized source code.

Running Locally

To view your site locally, you can simply double-click index.html. However, for the best experience (including smooth scrolling and internal links), use a local server like the Live Server extension in VS Code. Simply right-click index.html and select "Open with Live Server".

Deployment

Once you are ready to go public, upload your files to your hosting provider via FTP or cPanel File Manager. Your main file must be named index.html for it to be recognized as the homepage.

Common Pitfalls

Images not showing? Check the file paths in the <img> tags. Ensure the images are inside the assets/ folder as referenced.

Layout broken? Ensure you haven't renamed any CSS classes. The layout relies on specific class names defined in the style block.

2. Customization Guide

Make the website truly yours by tweaking the colors, fonts, and content. Most of the configuration is handled at the top of the index.html file.

How to Modify Colors

The site uses CSS variables for easy color management. Look for the :root block at the top of the stylesheet.

:root {
  --primary: #c84b2f;  /* Main Accent Color */
  --secondary: #2d7d6f; /* Secondary Theme Color */
  --bg: #faf8f5;       /* Light Mode Background */
  --text: #1a1008;     /* Body Text Color */
}
TODO: Locate the [data-theme="dark"] block to customize the dark mode appearance.

Google Fonts Integration

Change the typography by updating the HTML <link> in the <head> section. Once you add a new font, update the CSS variable:

--font-body: 'New Font Name', sans-serif;

Replacing Images

Replace existing images in the assets/ folder using the same names, or update the src attribute in HTML. Recommended size for Hero image: 1920x1080px.

Updating Text Content

All headings and paragraphs are found between <h1>, <h2>, and <p> tags. Use Ctrl+F to search for specific text you wish to change.

Best Practices

  • ✅ Always keep a backup of the original index.html before editing.
  • ✅ Use compressed images to keep page load speeds fast.
  • ❌ Don't remove the viewport meta tag; it will break responsiveness.

3. Page Structure

The site is organized into logical sections, making it easy to navigate and maintain. Each component is self-contained.

Home Page (index.html)

Navbar: Sticky navigation with Logo (left), Links (center), and Action buttons (right).
Hero Section: High-impact introduction with floating mosaic pieces and primary CTA buttons.
About Artist: Detailed biography area with a skill grid and unique "Years of Experience" badge.
Services: 3x2 Grid of service cards (Murals, Restorations, Workshops, etc.) with pricing info.
Portfolio Gallery: Categorized work showcase with high-res lightbox and category filters.
Process: 4-step walkthrough explaining the journey from consultation to installation.
Testimonials: Scrolling card section featuring client feedback and trust badges.
Contact: Integrated form for project inquiries with success toast notifications.

Login / Register

Standardized forms for client onboarding. These are standalone pages designed to sync with your backend (PHP/Node/Firebase).

Dashboard (Planned)

The login flow leads to a future dashboard where clients can track project progress, view invoices, and chat directly with the artist.
